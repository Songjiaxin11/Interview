克隆仓库: `git clone  
git@github.com:Songjiaxin11/Interview.git`  
切换分支: `git checkout branch-name` **一定要切换至个人分支**  
拉取分支: `git pull`  
添加修改: `git add .`  
提交修改: `git commit -m "message"`  
推送至分支: `git push`
